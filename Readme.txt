Packages:
1.) Mapping: Functional map created in map/myMap.pgm 
2.) pick_objects package: Creates a node - The robot travels to the desired pickup zone, displays a message that it reached its destination, waits 5 seconds, travels to the desired drop off zone, and displays a message that it reached the drop off zone.
3.)  add_marker package: The marker should initially be published at the pickup zone. After 5 seconds it should be hidden. Then after another 5 seconds it should appear at the drop off zone.\
4.) home_service.sh: runs 5 terminals to accomplish job :
Initially show the marker at the pickup zone.
Hide the marker once your robot reach the pickup zone.
Wait 5 seconds to simulate a pickup.
Show the marker at the drop off zone once your robot reaches it.
5.) my_robot package contains the Ishan.world. The complete package is used for any further future functionality.
6.)Images hold some screenshots
